# Proyecto E2E - SauceDemo

## Requisitos
- Node.js instalado
- Cypress instalado (npm install cypress --save-dev)

## Ejecución
1. Clonar el repositorio
2. Instalar dependencias con:
   npm install
3. Ejecutar pruebas con:
   npx cypress open
   (seleccionar saucedemo_purchase.cy.js)

## Notas
- El flujo automatiza login, agregar productos, checkout y validación del mensaje final.

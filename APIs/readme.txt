# Proyecto APIs - Demoblaze

## Requisitos
- Node.js instalado
- Cypress instalado (npm install cypress --save-dev)

## Ejecución
1. Clonar el repositorio
2. Instalar dependencias con:
   npm install
3. Ejecutar pruebas con:
   npx cypress run --spec "cypress/e2e/demoblaze_api.cy.js"

## Notas
- Se prueban los endpoints signup y login con casos positivos y negativos.
